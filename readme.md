# Smart Expense Tracker
A Python-based console application to help users track, manage, and analyze day-to-day expenses using basic programming and algorithmic techniques.

Overview of the Project
This project is designed as part of the VITyarthi flipped course **CSE1021 â€“ Introduction to Problem Solving and Programming**.

It demonstrates the practical use of lists, sets, tuples, dictionaries, loops, functions, conditionals, and basic algorithms by building a realistic and useful application.

The Smart Expense Tracker will allow the user to add expenses, view records, calculate monthly spending, find the highest expense, remove duplicates, and compute the Kth highest expense. It also provides a random expense generator for testing and demonstration purposes.
